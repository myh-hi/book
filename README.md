# 屹涵的有声书网站

一个现代化的有声书和PDF阅读网站，采用Apple Music风格设计。

## 功能特点

- 有声书播放功能
- PDF文件预览和阅读
- 收藏夹管理
- 书库浏览
- 设置页面自定义选项
- 响应式设计，适配不同设备

## 技术栈

- 前端：HTML5, CSS3, JavaScript (原生)
- PDF预览：PDF.js
- 音频播放：HTML5 Audio API

## 页面说明

- **首页**：有声书播放器
- **书库**：浏览所有有声书和PDF文件
- **收藏**：管理收藏的内容
- **设置**：自定义应用选项

## 特色功能

- PDF文件第一页作为封面预览
- 有声书播放控制
- 黑暗模式界面
- 自定义强调色
- 阅读进度追踪
